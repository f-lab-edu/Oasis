# 유저 회원 가입

## 기본 절차

### 회원가입 화면

1. 회원가입 의사 질의(client)
   - 기본 가입 : email, password 
   - 소셜 가입 : email
2. user_auth에 유저 생성

### 프로필 정보 생성

1. UserInfo가 존재하는지 확인 
2. 닉네임 유효성 검사 
3. 한줄소개 작성(client)
4. 관심 카테고리 선택(client)
5. user_info에 유저 생성


## 회원가입은 했으나 프로필 생성이 안되었을 경우

1. 로그인
2. UserInfo가 존재하는지 확인
3. 존재하지 않으면 프로필 생성 화면 이동
